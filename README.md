# linux 常用命令

```shell
ls [-]
-l 长格式
-a 显示隐藏文件
-r 逆序
-t 按照时间顺序
-R 递归显示
```

```shell
tar
-c 打包
-x 解包
-f 指定操作类型为文件
```
